# Price Predictor
Predict price of an article from image and other informations
<hr>

## Data:
* 99956 records

### Inputs:
* Date - ddmmyyyy
* Brand - string
  * 2 categories
* Model name - string
* Retail price in USD - float
* Release date - ddmmyyyy
* Size - float
* Region - string

### Output:
* Sale price in USD - float
